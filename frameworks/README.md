# Agentic Workflow Development Frameworks

Here we are trying to put together the list of frameworks that people are using in the neutron science community for developing agentic workflows.

| Framework | Type | Language | Key Strength | Best For | Abstraction Level | Visual Builder |
|-----------|------|----------|--------------|----------|-------------------|----------------|
| **LangChain** | Code-first | Python, JS/TS | Massive ecosystem & integrations | Complex, customizable LLM pipelines | Medium | ❌ |
| **LangGraph** | Code-first | Python, JS/TS | Stateful graph-based agent flows | Multi-agent, cyclical workflows | Medium-Low | Partial |
| **Strands Agents** | Code-first | Python | Simplicity & AWS-native | AWS-integrated agentic apps | High | ❌ |
| **n8n** | Low-code | Node.js | Visual workflow automation | Non-devs, automation pipelines | High | ✅ |
| **Dify** | Low-code | Python/TS | All-in-one LLM app platform | SaaS-style AI app deployment | High | ✅ |
